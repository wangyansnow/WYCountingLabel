# WYCountingLabelDemo

一款使用CoreAnimation实现动画数字变动效果

## Installation

- 1.cocopods
	* pod 'WYCountingLabel'
- 2.手动导入
	* 将`WYCountingLabel`整个文件夹拖入项目即可
